    StayDrippy

    Languages: TypeScript, JavaScript and CSS

    Stay Drippy is an AI-powered fashion assistant designed to help users win "Dress to Impress" games on Roblox. By leveraging OpenAI, the program analyzes a user's wardrobe based on uploaded images and descriptions.

    Mood Tracker

    Languages: JavaScript, HTML, CSS and Python

    The Mood Tracker project is a web application designed to help users monitor their emotions and moods over time. By leveraging databases and intelligent logic, the application delivers personalized mood analysis, offering valuable insights into emotional patterns and well-being.

    Parking Efficiency Algorithm

    Language: C

    The Parking Efficiency Algorithm project is a C-based program designed to optimize parking lot management on campus, reducing congestion and improving the overall parking experience.

    Motor-Powered Airplane

    Worked collaboratively with a multidisciplinary team of five to design and refine a motor-powered airplane. As the lead of the electrical engineering team, implemented advanced circuit techniques and conducted simulations using LTspice to ensure efficiency and reliability in the aircraft’s electrical systems.

    Micromouse Algorithm

    The Micromouse project, developed in collaboration with IEEE, involved designing a virtual maze-solving robot using a programmed labyrinth simulator. The project leveraged a Python-based algorithm to control the micromouse’s movements, enabling it to navigate and optimize its path through the maze efficiently. By implementing pathfinding techniques such as flood-fill and A* search, the algorithm continuously adapted to the maze’s layout, improving its decision-making process.

    Combinational Logic Circuits

    This project explores the use of RAM (Random Access Memory) and ROM (Read-Only Memory) as practical implementations for combinational logic circuits. By configuring memory elements to store and retrieve specific data patterns, the experiment demonstrates how complex logic functions can be realized without traditional gate-based design.
